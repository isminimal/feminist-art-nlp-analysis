# Exploring Feminist Art-Historical Texts with NLP

This project explores how feminist art-historical texts articulate interpretive concerns using Natural Language Processing (NLP) methods.

Rather than aiming at prediction or classification, the project investigates how different NLP techniques model theoretically dense humanities texts and what kinds of structures they reveal or fail to capture.

## Dataset

- 9 scholarly texts (PDF)
- Domain: Feminist art history and visual culture
- Language: English
- Small, theory-rich corpus

## Methods

- TF-IDF vectorization (lexical analysis)
- Topic modeling (LDA)
- Word embeddings (Word2Vec)
- Semantic similarity (cosine similarity)

## Approach

The project is exploratory and method-oriented. It compares how different NLP representations structure interpretive texts and highlights the limits of computational abstraction in humanities research.

## Key Insight

This project explores how NLP methods can be used to model discursive structures and conceptual relationships in feminist art-historical texts, foregrounding their potential as context-aware instruments for engaging with interpretive, historically situated discourse. Such extensions require careful attention to data access, interpretive modeling frameworks, and underlying epistemological assumptions.

## Context

Developed as part of coursework in Machine Learning and NLP in Digital Humanities.
